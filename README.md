# The Gdtu-Block-Explorer
The Gdtu Block Expolorer

## How to start?
```bash
$ npm install
$ GDTU_RPC_URL=<YOUR_RPC_URL> npm start
```

## What features are there?
This app use [`web3.js`](https://github.com/c88032111/web3.js/) library.

* Block info search 
* Transaction info search 
* Address info search
